# quic-traffic-simulator

## Usage

```bash
# clone all
$ cd path/to/your/workspace
$ git clone -b dev --single-branch https://github.com/LouisLU9911/s2n-quic.git
$ git clone https://github.com/LouisLU9911/quic-traffic-simulator.git

# build
$ make build

# apply tc
$ make tc

# docker-compose up
$ make
```
